# LockDevice · 专注锁定

帮你「管住自己」的 Windows 桌面小工具：设定时长后锁定电脑，防止学习时忍不住玩。
提供 **软 / 硬两种锁定方式**，可安装到本机（开机自启、免 UAC），也可免安装绿色运行。

## 下载

到 [Releases](../../releases) 下载最新的 `LockDevice-<版本>.exe`，双击即用（自带解释器，无需安装 Python）。

> 仅支持 Windows。**首次务必先用 1–3 分钟短时长试**，确认行为符合预期再加长。

## 两种锁定方式

### ① 全屏锁定（软）
全屏黑幕盖住屏幕，只有 **时钟 + 大倒计时 + 「🌙 息屏」「⏻ 关机」两个按钮**；拦截
`Win / Alt+Tab / Alt+F4 / Ctrl+Esc` 等逃逸热键，到点自动解锁。两个可选开关：

- **🛡 防止结束进程**：双进程互守（分离进程树），任务管理器结束其一，另一个立即拉起。
- **🚫 禁用任务管理器**：锁定期间任务管理器打不开（绿色模式会弹一次 UAC 提权）。
- **🌙 息屏**：只熄灭显示器防烧屏（不锁屏），点击后延迟 1 秒再息屏；动一下鼠标即亮屏，锁仍在。

### ② 定时关机（硬）
立即关机，并在设定时间内**反复自动关机**，到点自动解除。两个可选开关：

- **🛡 加强防护 · 登录前关机**：`SYSTEM + 开机触发`，登录前就关、更硬（绿色模式弹一次 UAC）。
  默认「登录后关机」用当前用户·最低权限任务，标准用户**免 UAC**。
- **⚡ 开始后立即关机**：不留 25 秒缓冲、开始即关，防止在倒计时窗口内清数据逃脱。

> 🆘 **安全阀**：万一异常反复关机，开机进 **安全模式**（计划任务不触发）→ 运行本程序点
> 「解除 / 卸载」，或执行 `schtasks /delete /tn LockDevice_Shutdown /f`。

## 安装 / 绿色运行 / 更新

| 操作 | 说明 |
|------|------|
| 🟢 绿色运行 | 免安装直接用，基本不留痕迹（锁定期间重启自动恢复，结束自动清理）。 |
| ⬇️ 安装到本机 | 一次管理员授权 → 开机自启 + 此后免 UAC；默认装到 `%ProgramFiles%\LockDevice` 并上 ACL 锁防篡改。 |
| 🔄 更新 | 在已安装机器上双击新版 exe：免 UAC 打开已安装版 + 弹「发现更新」窗；点更新原地升级、保留设置。 |
| ❌ 卸载 / 🧹 清除数据 | 删除任务 / 自启 / 文件；绿色残留一键清理。 |

## 从源码运行 / 打包

```bash
pip install customtkinter
python lock_device.py     # 运行
python build.py           # 打包 → dist\LockDevice-<版本>.exe
```

**发布新版**：改 `lock_device.py` 顶部的 `VERSION` 并在 `CHANGELOG` 加一条，推送到 `main`——
GitHub Actions 会自动构建并发布 Release（发布说明取自 `CHANGELOG`）。

## 插件（1.4.0+）

功能以插件形式扩展：放在 `plugins/` 目录，程序启动**自动识别加入**，主页每个插件一个按钮 + 统一「⚙ 设置」页；打包默认包含所有插件。插件与本体**共用配置**、且**只写后端 + 声明前端**（界面由本体渲染，故日后换 Qt 界面插件零改动）。

- 内置 **⏰ 自动锁机**：设定每天固定时间自动全屏锁定一段时长（后续支持跳过节假日 / 按周 / 按月）。
- 想自己写插件？见 [`docs/PLUGINS.md`](docs/PLUGINS.md)。

> 🗺 路线图：界面计划迁移到 Qt 做得更精美（插件的声明式前端已为此铺路）。

## 诚实的边界

这是你自己的电脑，用户态无法 100% 防杀；**安全模式**始终是留给你的退路。工具的意义是
**抬高分心的门槛**，而非绝对不可逾越。免 UAC 靠的是安装时你亲自授权的「最高权限计划任务」，
且只运行受 ACL 保护的可信副本，不构成通用提权后门。
